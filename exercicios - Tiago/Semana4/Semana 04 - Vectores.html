Geração de Números Aleatórios

O Java oferece a classe estática Math, que disponibiliza várias funções para efetuar cálculos. Entre essas funções existe uma para gerar números aleatórios Math.random()), que devolve um número decimal (double) aleatório no intervalo [0.0, 1.0[. Com este valor podemos sortear inteiros da seguinte forma:

double r = Math.random(); // r estará no intervalo [0.0, 1.0[
int i = (int) (r * max); // i estará no intervalo [0, (max - 1)]
