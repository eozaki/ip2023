A principal novidade neste conjunto de exercícios é a noção de procedimento. Um procedimento recebe referências para, por agora, vetores, os quais pode manipular, ao contrário das funções que produzem um resultado que não altera os argumentos. Tal como com as funções, para testar um procedimento que tem vetores como parâmetros, pode ser criado um procedimento de teste sem parâmetros onde é definido um vetor que será o argumento a ser passado ao procedimento que se pretende testar. Por exemplo, para testar o procedimento swap() pedido no exercício B desta aula, podemos definir um procedimento de teste como se pode ver abaixo.

static void swap(int i, int j, char[] v) { ... }

static void testSwap() {
    char[] v = {'a', 'c', 'e', 'g', 'i'};
    swap(2, 3, v);
}
